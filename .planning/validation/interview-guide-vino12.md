## Interview Script

### Pre-interview Checklist

- [ ] Opnameapparatuur getest
- [ ] Toestemmingsformulier klaar
- [ ] Notitieblad/template klaar
- [ ] Incentive bij de hand
- [ ] Rustige locatie geregeld

### Opening (5 min)

```
"Bedankt dat je tijd hebt gemaakt. Ik doe onderzoek naar hoe mensen
in Nederland wijn kopen en kiezen. Er zijn geen goede of foute
antwoorden — ik wil gewoon jouw persoonlijke ervaring begrijpen.

Het gesprek duurt ongeveer 45 minuten. Alles wat je vertelt is
anoniem. Mag ik het gesprek opnemen voor mijn aantekeningen?"
```

**Let op:** Noem VINO12 NIET. Noem geen product-idee.

### Blok 1: Wijngedrag in het Algemeen (10 min)

**Doel:** Context begrijpen — hoe kopen en kiezen ze nu wijn?

```
"Vertel eens, hoe ziet een typische wijnkoop eruit voor jou?"
```

Doorvragen:

- "Hoe vaak koop je wijn? Per week, maand?"
- "Waar koop je meestal wijn?" (supermarkt, slijter, online, vakantie)
- "Hoeveel geef je gemiddeld uit per fles?"
- "Koop je voor jezelf, voor een diner, als cadeau?"
- "Hoe kies je welke wijn je koopt?" (label, prijs, streek, druif, advies)

### Blok 2: Laatste Aankoop — Timeline (10 min)

**Doel:** Specifiek reconstrueren — wat triggerde de aankoop?

```
"Kun je me vertellen over de laatste keer dat je wijn kocht?
Loop me helemaal erdoorheen, van het moment dat je dacht
'ik wil wijn' tot het moment dat je een fles opende."
```

Doorvragen:

- "Wanneer was dat precies?" (dag, gelegenheid)
- "Waar was je toen je besloot wijn te kopen?"
- "Hoe lang duurde het kiezen?"
- "Was er iemand bij betrokken?" (partner, vrienden)
- "Wat was het uiteindelijke bedrag?"
- "Was je tevreden met je keuze? Waarom wel/niet?"

### Blok 3: Push — Frustraties (10 min)

**Doel:** Wat is er niet goed aan de huidige manier van wijn kopen?

```
"Als je denkt aan hoe je nu wijn koopt — wat zou je
graag anders zien? Wat frustreert je?"
```

Doorvragen:

- "Heb je weleens een fles gekocht die tegenviel? Vertel."
- "Wat is het moeilijkste aan het kiezen van wijn?"
- "Hoe vaak koop je dezelfde wijn vs iets nieuws?"
- "Heb je weleens het gevoel dat je teveel betaalt?"
- "Wat doe je als je wijn wilt voor een speciale gelegenheid?"

**Mom Test check:** Luister naar wat ze DEDEN, niet wat ze ZOUDEN doen.

### Blok 4: Pull — Ideale Situatie (5 min)

**Doel:** Wat trekt hen aan in een betere oplossing?

```
"Als je een toverstaf had en het wijn-kopen kon veranderen
— hoe zou het er dan uitzien?"
```

Doorvragen:

- "Wat zou je het meest helpen bij het kiezen?"
- "Zou je iemands advies vertrouwen? Van wie?"
- "Hoe belangrijk is het om nieuwe wijnen te ontdekken vs favorieten?"

### Blok 5: Online Wijn — Ervaring (10 min)

**Doel:** Specifiek online koopgedrag en barrières begrijpen.

```
"Heb je weleens wijn online besteld?"
```

**Als JA:**

- "Loop me erdoorheen — welke site, waarom die, hoe was de ervaring?"
- "Wat beviel je wel/niet?"
- "Zou je het opnieuw doen? Waarom?"
- "Hoe vergeleek je de ervaring met de slijter/supermarkt?"

**Als NEE:**

- "Heb je het overwogen? Wat hield je tegen?"
- "Wat zou je overtuigen om het te proberen?"
- "Koop je andere dingen online die je ook in de winkel kunt kopen?"

Doorvragen (altijd):

- "Wat vind je van wijnbeschrijvingen online? Helpen die?"
- "Zou je een wijn bestellen die je nooit geproefd hebt?"
- "Wat denk je van bezorgd krijgen vs zelf ophalen?"

### Blok 6: Commitment Escalation (5 min)

**Doel:** Test de echte motivatie met toenemend commitment.

```
Niveau 1 (Tijd):
"Ik ben bezig met een nieuw project rondom wijn.
Mag ik je over 2-3 weken nog een keer spreken?"

Niveau 2 (Reputatie):
"Ken je nog andere wijnliefhebbers die ik zou kunnen spreken?"

Niveau 3 (Actie):
"Als ik een mailinglijst start met wijn-tips en aanbiedingen,
zou je je willen aanmelden?"
```

### Afsluiting (2 min)

```
"Is er iets over wijn kopen dat ik niet gevraagd heb maar
dat je belangrijk vindt?"

"Heel erg bedankt voor je tijd. Hier is je [incentive]."
```

| Keuzestrategie | Prijs / Streek / Druif / Advies / Label |
| Interesse in curatie | 1-10 |

**Assumption validatie per interview:**
| Assumption | Gevalideerd? | Evidence |
|-----------|-------------|---------|
| #1 Merkvertrouwen nieuw merk | Ja/Nee/Onduidelijk | [citaat] |
| #3 Betalen zonder proeven | Ja/Nee/Onduidelijk | [citaat] |
| #5 Curatie als differentiator | Ja/Nee/Onduidelijk | [citaat] |

### Na 10 Interviews

**Stap 1:** Update confidence scores in assumption-map.md
**Stap 2:** Cluster jobs in categorieën
**Stap 3:** Formuleer opportunity statements
**Stap 4:** Input voor landing page test (welke value prop resoneert het meest?)

### Verwachte Job Statements (hypothesen)

1. "Help me ontdekken welke wijn ik lekker vind zonder 50 flessen te kopen"
2. "Help me indruk maken op mijn gasten met een bijzondere wijn"
3. "Help me goede wijn vinden zonder uren research te doen"
4. "Help me een betrouwbare wijn te kiezen voor een speciale gelegenheid"

## Red Flags Checklist

Na elke interview, check:

- [ ] Respondent koopt daadwerkelijk ≥1x/maand wijn
- [ ] Specifieke gebeurtenissen beschreven (niet hypothetisch)
- [ ] Geen sociaal wenselijke antwoorden gedetecteerd
- [ ] Mom Test regels gevolgd (niet gepitcht)
- [ ] Audio-opname succesvol
- [ ] Commitment level bereikt (minimaal niveau 1)
