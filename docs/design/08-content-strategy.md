# VINO12 Content Strategy

> Voice, Tone, en Microcopy Richtlijnen

---

## Content Principles

| Principle   | Description                                  | Example                                                     |
| ----------- | -------------------------------------------- | ----------------------------------------------------------- |
| **Direct**  | Geen omwegen, geen jargon. Zeg het gewoon.   | "Rode wijn uit Piemonte" niet "Een sierlijk rood pareltje"  |
| **Eerlijk** | Geen superlatieven tenzij verdiend.          | "Populair bij onze klanten" niet "De beste wijn ter wereld" |
| **Warm**    | Direct maar niet koud. Wijn is gezelligheid. | "Fijn dat je er bent" niet "Welkom bij VINO12 B.V."         |

---

## Voice & Tone

### Brand Voice

| Attribute   | We Are                        | We Are Not                        |
| ----------- | ----------------------------- | --------------------------------- |
| **Direct**  | Helder, beknopt, to-the-point | Omslachtig, wollig, vaag          |
| **Eerlijk** | Transparant, onderbouwd       | Overdreven, bewerend, pretentieus |
| **Warm**    | Uitnodigend, persoonlijk      | Kil, zakelijk, afstandelijk       |
| **Kennis**  | Informed maar toegankelijk    | Snobistisch, neerbuigend          |

### Tone per Context

| Context              | Tone                            | Example                                                                       |
| -------------------- | ------------------------------- | ----------------------------------------------------------------------------- |
| Product beschrijving | Informatief + warm              | "Vol van smaak, perfect bij pasta of een stevige kaas."                       |
| CTA                  | Direct + uitnodigend            | "Bestel nu" / "Voeg toe"                                                      |
| Error                | Behulpzaam + niet beschuldigend | "Betaling niet gelukt. Probeer het opnieuw of kies een andere betaalmethode." |
| Empty state          | Licht + motiverend              | "Je winkelwagen is nog leeg. Ontdek onze wijnen."                             |
| Success              | Bevestigend + warm              | "Bestelling geplaatst! We pakken je wijn zorgvuldig in."                      |

## Microcopy Library

### UI Patterns

| Context             | Copy                                                             |
| ------------------- | ---------------------------------------------------------------- |
| Loading             | "Even laden..."                                                  |
| Empty cart          | "Je winkelwagen is nog leeg."                                    |
| Add to cart success | "Toegevoegd aan je winkelwagen"                                  |
| Remove from cart    | "Verwijderd"                                                     |
| Out of stock        | "Helaas uitverkocht"                                             |
| 404                 | "Deze pagina bestaat niet. Misschien een goed glas wijn zoeken?" |
| Checkout success    | "Bestelling geplaatst! Ordernummer: {{nr}}"                      |
| Age gate            | "Ben je 18 jaar of ouder?"                                       |
| Cookie consent      | "We gebruiken cookies voor een betere ervaring."                 |

### Button Labels

| Action           | Label           | Don't Use           |
| ---------------- | --------------- | ------------------- |
| Primary purchase | "Bestel nu"     | "Koop", "Purchase"  |
| Add to cart      | "Voeg toe"      | "In winkelwagen"    |
| Remove           | "Verwijder"     | "Delete", "X"       |
| Back             | "Terug"         | "Go back"           |
| Submit review    | "Plaats review" | "Submit", "Verzend" |
